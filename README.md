## Instrucciones
1. Para el server en python necesitamos instalar: 
`pip install flask` y 
`pip install flask-cors`

y ejecutar servidorweb.py


2. Para el cliente necesitamos escribir: 
`npm install`
   y ejecutar:
`npm run dev`
